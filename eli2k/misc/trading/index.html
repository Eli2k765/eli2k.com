Technical Analysis

  • Swing
    This is a swing trading strategy for triple-leveraged ETFs using PineScript in TradingView. The entry condition is based on an exponential moving average cross of the closing price if the consumer confidence index (CCI) is above or below a threshold. This helps identify momentum shifts in the market. The exits use a combination of a relative strength index (RSI) reaching overbought or oversold status if CCI is above or below a threshold to determine when a swing has likely ended. There is an option to take a reverse position if this exit condition is met. If this condition is not met, then the strategy will use the normal entry condition.
