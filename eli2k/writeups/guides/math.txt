1. Basic arithmetic: Addition, Subtraction, Multiplication, Division
2. Algebra: 
    - Exponents: a ^ b
    - Order of Operations: PEMDAS (Parentheses, Exponents, Multiplication and Division, Addition and Subtraction)
    - Factoring: a^2 - b^2 = (a+b)(a-b)
    - Quadratic Formula: x = (-b ± sqrt(b^2 - 4ac)) / (2a)
3. Geometry: 
    - Pythagorean Theorem: a^2 + b^2 = c^2
    - Area of a Triangle: A = (1/2)bh
    - Surface Area of a Sphere: A = 4πr^2
    - Volume of a Cylinder: V = πr^2h
4. Trigonometry: 
    - Sine, Cosine, and Tangent: sinθ, cosθ, tanθ
    - Trigonometric Identities: sin^2θ + cos^2θ = 1
    - Law of Cosines: c^2 = a^2 + b^2 - 2ab cos(θ)
    - Law of Sines: a/sin(A) = b/sin(B) = c/sin(C)
5. Statistics: 
    - Mean: (x1 + x2 + ... + xn) / n
    - Standard Deviation: sqrt( (1/n) * ((x1 - x̄)^2 + (x2 - x̄)^2 + ... + (xn - x̄)^2) )
    - Normal Distribution: f(x) = (1 / sqrt(2πσ^2)) * e^(-(x-μ)^2 / 2σ^2)
    - Correlation: r = (n∑xy - (∑x)(∑y)) / sqrt((n∑x^2 - (∑x)^2)(n∑y^2 - (∑y)^2))
6. Number Theory: 
    - Prime Numbers: Numbers that are only divisible by 1 and themselves.
    - Greatest Common Divisor: gcd(a,b)
    - Divisibility Rules: Rules for determining if a number is divisible by another number without performing long division.
    - Modular Arithmetic: (a + b) mod n = ((a mod n) + (b mod n)) mod n
7. Discrete Math: 
    - Sets: A collection of distinct objects.
    - Propositional Logic: A branch of logic that studies propositions and their logical relationships.
    - Combinatorics: The study of the number of ways to choose or arrange objects.
    - Graph Theory: The study of graphs and their properties.
8. Linear Algebra: 
    - Matrices: A rectangular array of numbers.
    - Vectors: A quantity that has both magnitude and direction.
    - Eigenvalues and Eigenvectors: A scalar and a non-zero vector that satisfy the equation Av = λv.
    - Systems of Linear Equations: A set of equations that can be expressed in the form Ax = b.
9. Calculus: 
    - Derivatives: The rate of change of a function with respect to its input variable.
    - Integrals: The area under a curve.
    - Limits: The value that a function approaches as its input variable approaches a certain value.
    - Taylor Series: A representation of a function as an infinite sum of terms calculated from the function's derivatives at a single point.
10. Abstract Algebra: 
    - Groups: A set of elements with a binary operation that satisfies four properties.
    - Rings: A set of elements with two binary operations that satisfy certain properties.
    - Fields: A set of elements with two binary operations that satisfy certain properties.
    - Galois Theory: The study of the symmetry of roots of polynomials.
11. Differential Equations: 
    - Ordinary Differential Equations: An equation that relates a function to its derivatives with respect to a single variable.
    - Partial Differential Equations: An equation that relates a function to its partial derivatives with respect to multiple variables.
    - Separation of Variables: A method for solving certain types of partial differential equations.
    - Laplace Transforms: A method for solving certain types of differential equations.
12. Real Analysis: 
    - Continuity: A property of a function that means it has no breaks or gaps.
    - Differentiability: A property of a function that means it has a derivative at every point in its domain.
    - Riemann Integral: A method for defining the integral of a function over an interval.
    - Lebesgue Integral: A method for defining the integral of a function over any set.
13. Complex Analysis: 
    - Complex Numbers: Numbers with a real part and an imaginary part.
    - Analytic Functions: Functions that are complex differentiable.
    - Cauchy-Riemann Equations: A set of partial differential equations that analytic functions satisfy.
    - Residue Theorem: A method for calculating certain integrals using complex analysis.
